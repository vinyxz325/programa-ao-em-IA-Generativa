# 🚀 Curso de Programação em IA Generativa

Bem-vindo ao repositório do curso de **Programação em Inteligência Artificial Generativa**.
Aqui você encontrará materiais, aulas, exemplos práticos e projetos voltados para o desenvolvimento de aplicações utilizando IA moderna.

---

# 📚 Conteúdo do Curso

## 🔹 Preparação

* Introdução ao ambiente
* Configuração de ferramentas
* Setup para desenvolvimento com IA
* Conceitos iniciais

## 🔹 Módulos do Curso

### Aula 1 — Inteligência Artificial

* Conceitos fundamentais
* História da IA
* Tipos de IA
* Aplicações atuais

### Aula 2 e 3 — Aprendizado de Máquina

* Machine Learning
* Modelos supervisionados e não supervisionados
* Treinamento de modelos
* Avaliação de desempenho

### Aula 4, 5 e 6 — Redes Neurais Artificiais

* Perceptron
* Deep Learning
* Funções de ativação
* Backpropagation
* Frameworks modernos

### Aula 7 e 8 — Processamento de Linguagem Natural (NLP)

* Tokenização
* Embeddings
* Transformers
* LLMs (Large Language Models)
* Chatbots e assistentes inteligentes

### Aula 9, 10 e 11 — Visão Computacional

* Processamento de imagens
* Classificação
* Detecção de objetos
* OCR
* Modelos multimodais

### Aula 12 e 13 — Modelos Personalizados

* Fine-tuning
* RAG (Retrieval-Augmented Generation)
* Agentes de IA
* Deploy de modelos
* Integração com APIs

# ⚙️ Configuração do Ambiente

## 1️⃣ Clone o repositório

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

## 2️⃣ Acesse a pasta

```bash
cd seu-repositorio
```

## 3️⃣ Crie um ambiente virtual

### Linux/Mac

```bash
python -m venv venv
source venv/bin/activate
```

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

## 4️⃣ Instale as dependências

```bash
pip install -r requirements.txt
```
